# Demo SFA Go

Отдельный статический репозиторий для проекта `Demo SFA Go`.

## Структура

- `index.html` - единая точка входа в проект
- `meetings/` - канонический экран встреч
- `debit-card/` - отдельный продуктовый флоу дебетовой карты
- `sms/` - standalone-макет SMS
- `meeting-transfer/` - флоу переноса встречи
- `acts/` - модуль `Акты`
- `no-answer/` - сценарий `Не дозвонился`
- `shared/meetings-store.js` - общий store состояния встреч
- `PROJECT_MAP.md` - карта модулей и связей

## Локальный запуск

Самый простой способ: открыть `index.html` в браузере.

Если нужен локальный HTTP-сервер:

1. Открыть PowerShell в корне репозитория
2. Выполнить `npx serve .`

## Публикация

Репозиторий подготовлен под отдельный GitHub Pages сайт.

Для привязки отдельного домена:

1. Добавить файл `CNAME` в корень репозитория
2. Записать в него целевой домен одной строкой, например `demo.example.com`
3. В GitHub открыть `Settings -> Pages`
4. Включить публикацию из ветки `main` и папки `/ (root)`

После этого проект будет открываться напрямую из корня домена без общей витрины других макетов.
